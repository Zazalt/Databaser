Databaser is a PHP library/package to 'translate' a database to array.

Requirements
---------------
* php >= 5.4.0

Packagist Dependencies
---------------
* [zazalt/omen](https://github.com/zazalt/omen)

Installation
---------------
With composer:
``` json
{
	"require": {
		"zazalt/databaser": "dev-master"
	}
}
```
